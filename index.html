📚 教学说明（跟着关卡学习）

  1. 第1关：均值差（Difference in Means）
    你会观察到处理组与对照组的平均值差：ATÊ = mean(Y|D=1)-mean(Y|D=0)。
    在真正随机分组下，这个估计是无偏的。
  2. 第2关：非随机分组与混杂偏误
    我们让能力高的个体更容易进入处理组。此时均值差会混入“能力差异”导致偏误。
    你会看到即使真实效应固定，估计也可能系统偏高或偏低。
  3. 第3关：回归分析（控制变量）
    用线性模型：Y = β₀ + β₁D + β₂X + ε。
    其中 X 是可观测混杂变量（如能力）。在模型可用条件下，β₁ 比裸均值差更接近真实处理效应。

🧪 你可以做的“列举实验”

  • 固定 τ=2，把 N 从 20 增大到 300，观察置信区间如何变窄。
  • 固定 N=80，把 σ 从 2 提到 20，观察统计显著性如何下降。
  • 第2关把混杂强度从 0 拉到 5，比较偏误放大过程。
  • 第3关对比“均值差”和“回归 β₁”谁更接近真实 τ。
