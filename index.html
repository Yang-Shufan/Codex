🧭 先学后玩：4 个最关键知识点

知识点1：均值差是什么？

把“用药组平均分”减去“没用药组平均分”，就是最直观的效果估计：ATÊ = mean(Y|D=1)-mean(Y|D=0)。

知识点2：为什么样本量重要？

N 小时，运气成分大；N 大时，估计更稳，置信区间更窄。

知识点3：混杂偏误是什么？

如果“更强的人更容易进用药组”，你看到的提升不一定来自药本身，可能是人本来就更强。

知识点4：回归在干嘛？

用 Y = β₀ + β₁D + β₂X + ε 控制能力 X，尽量把“药效（β₁）”和“能力差异（β₂）”分开。

🧪 推荐实验清单（从简单到深入）

  1. 第1章：固定 τ=2，把 N 从 20 拉到 300，感受“更稳”。
  2. 第2章：固定 N=80，调大 σ，观察置信区间变宽。
  3. 第3章：调大偏心分组强度，观察均值差如何系统偏离 τ。
  4. 第4章：对比 |ATÊ-τ| 与 |β₁-τ|，看回归是否修正偏误。
