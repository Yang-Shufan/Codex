# 统计冒险学院（重构版）

这是一个**完全重构后的多页面关卡游戏**，针对“统计学小白”设计。
不再是单页堆参数，而是每个知识点独立成关，先任务后解释。

## 核心改动

- ✅ 从单页改成 **章节地图 + 4 个独立关卡页面**
- ✅ 默认不要求用户先调一堆参数
- ✅ 每关只保留 1~2 个关键决策，降低认知负担
- ✅ 通过徽章与解锁机制推进学习

## 目录结构

- `index.html`：章节地图（查看解锁进度）
- `chapters/ch1.html`：均值差侦察队
- `chapters/ch2.html`：样本量工坊
- `chapters/ch3.html`：混杂偏误法庭
- `chapters/ch4.html`：回归救援战
- `app.js`：共享状态与统计计算
- `styles.css`：统一样式

## 运行方式

### 方式1（推荐）
```bash
python3 -m http.server 8000
```
然后打开：

`http://localhost:8000/index.html`

### 方式2
直接双击 `index.html`（部分浏览器环境可能有本地限制，推荐方式1）。

## 学习路径（建议）

1. 第1关先练直觉：均值差到底在比较什么。
2. 第2关理解“为什么 N 大更稳”。
3. 第3关识别混杂：不是“看起来有效”就是真的因果。
4. 第4关比较 ATÊ 与回归 β₁，理解控制变量的价值。
