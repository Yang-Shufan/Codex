# 实验训练营：统计小白冒险版

这是一个面向**统计学新手**的可玩教学游戏（单文件 `index.html`）。
目标是用“任务闯关”的方式，理解从均值差到回归分析的核心思想。

## 你会学到什么（先给结论）

1. **均值差**：最直观的处理效应估计。
2. **样本量/噪声**：为什么有时结果不稳定。
3. **混杂偏误**：为什么“看起来有效”不等于“真的有效”。
4. **回归控制变量**：如何把药效与人群差异分开。

## 怎么运行

### 方法1：直接打开
下载仓库后，双击 `index.html` 用浏览器打开。

### 方法2：本地服务器（推荐）
```bash
python3 -m http.server 8000
```
然后访问：
```text
http://localhost:8000/index.html
```

## 新手建议玩法（不要跳）

- 每章先点击“一键新手参数”，跑 2~3 次建立直觉。
- 按章做任务，不要直接跳到最后一章。
- 每章拿到徽章后再继续下一章。

## 4章结构

- 第1章：认识均值差（先理解估计量）
- 第2章：样本量与噪声（理解不确定性）
- 第3章：混杂偏误（理解非随机分组风险）
- 第4章：回归修正（比较 `ATÊ` 与 `β₁`）

## 推荐列举实验

1. 第1章固定 `τ=2`，让 `N` 从 20 到 300。
2. 第2章固定 `N=80`，让 `σ` 从 2 到 18。
3. 第3章调高偏心分组强度，观察偏误变化。
4. 第4章对比 `|ATÊ-τ|` 与 `|β₁-τ|`。
