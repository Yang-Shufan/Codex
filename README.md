# 实验设计学院：从均值差到回归分析

一个可直接打开游玩的教学小游戏（单文件 `index.html`），目标是让学习者通过反复试验，理解：

1. 均值差估计（Difference in Means）
2. 随机与非随机分组的偏差差异
3. 回归控制变量对偏误修正的作用

## 快速开始

直接在浏览器打开 `index.html` 即可。

## 玩法与教学目标

- **第1关**：随机实验，体会均值差估计在随机化下的无偏性。
- **第2关**：非随机分组，观察混杂导致的偏误。
- **第3关**：使用回归模型 `Y = β0 + β1D + β2X + ε`，比较 `β1` 与简单均值差对真实效应 `τ` 的贴近程度。

## 建议教学活动（列举实验）

- 固定 `τ=2`，逐步放大样本量 `N`，观察置信区间变化。
- 固定 `N`，提高噪声 `σ`，观察估计不确定性变化。
- 在第2关调高混杂强度，记录偏误变化方向和幅度。
- 在第3关对比：`|ATÊ-τ|` 与 `|β1-τ|`。
