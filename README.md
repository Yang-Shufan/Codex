# 列表实验学院（系统进阶版）

这版在上一轮基础上继续增强：
- 知识体系更完整：从动机、估计、假设、到多元分析与设计决策。
- 入门更平滑：每关先给目标，再做1~2个核心交互，不要求一上来理解全部参数。
- 游戏性更强：新增星星、徽章、经验、预算挑战与毕业关。

## 课程结构（5关）

1. **第1关：列表实验是什么**
   - 对比：真实比例 / 直接提问 / 列表实验估计
   - 目标：直观看到社会期望偏误下列表实验的价值
2. **第2关：差均值估计与抽样分布**
   - 目标：理解“单次波动、长期居中”
3. **第3关：关键假设与违背检测**
   - 目标：操控设计效应、谎报、天花板/地板效应并观察诊断信号
4. **第4关：多元分析入门**
   - 目标：对比 DIM 与控制协变量后的估计误差
5. **第5关：实验设计挑战赛**
   - 目标：在预算约束下优化误差，完成毕业判定

## 运行

```bash
python3 -m http.server 8000
```
打开：`http://localhost:8000/index.html`

## 文件

- `index.html`：课程地图与解锁
- `app.js`：仿真引擎与进度状态
- `styles.css`：统一样式
- `chapters/ch1.html` ~ `chapters/ch5.html`：五个关卡

第4关是“多元分析教学入口”的简化版，帮助理解为什么文献会从均值差进一步走向 NLS/ML/EM 等更完整方法。
